About EdNet

In this rapidly changing world, new material and information are constantly being released with the intention of bettering our knowledge in certain topics. Thus, it only makes sense to make sure that all professors teach the best material that they can to help prepare upcoming students and future professionals for the real world. The product that aims to achieve this goal EdNet which is a central hub for sharing the highest quality materials for education.

EdNet is a web application that serves as a unified collection of education materials for instructors to use to their advantage. With these applications, a user will be able to register and login to their own account where they can create/edit their own learning modules as well as view and improve other ones. All modules consist of basic information that is uploaded by a user about the course and could include any links or files needed for it. A user can also search for modules based on categories like the type of module it is or the area it covers. Overall, these web applications will create an open-source environment where people all around can access/add to tested and enhanced learning material as well as contribute their own.
